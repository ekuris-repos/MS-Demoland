# AI Pair Programming — The Copilot Awakens — Class Notes

> **Track:** Developer | **Level:** Beginner | **Duration:** 30 minutes

## Delivery Notes

- **Speaker notes toggle:** Press N during the slide presentation to show or hide speaker notes below each slide. A "Speaker Notes ON" indicator appears at the bottom of the screen when notes are visible. Notes are also visible in print view.
- **Pacing:** Slides 1 through 10 (setup) should take approximately 8 minutes. The bulk of time goes to the live demo and lab walkthrough (slides 11 through 23).
- **Audience warm-up:** Ask "Who has already tried Copilot?" to gauge experience. Tailor depth accordingly.
- **Ghost text demo:** Slow down here — this is the "wow moment" for most attendees. Type the comment slowly, then pause to let the suggestion appear.
- **Chat commands:** Do `/explain` live on audience-suggested code if time allows.
- **Inline chat vs sidebar chat:** Clarify the difference — `Ctrl+I` (quick inline edit) vs `Ctrl+Shift+I` (full conversation panel).
- **Enterprise audience:** Emphasize that Business/Enterprise plans have no code retention for training and include IP indemnity.
- **Close strong:** End with the keyboard shortcuts slide — it's the most actionable takeaway.
