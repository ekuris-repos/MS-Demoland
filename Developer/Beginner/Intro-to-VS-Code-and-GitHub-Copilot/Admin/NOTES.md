# Intro to VS Code and GitHub Copilot — Class Notes

> **Track:** Developer | **Level:** Beginner

## Delivery Notes

_Tips for the presenter: pacing, emphasis areas, common audience questions._

- This is the absolute first course in the Developer track. Assume zero prior experience with VS Code or Copilot.
- Plan for about 10 minutes on VS Code (Slides 4–15) and 15 minutes on Copilot (Slides 16–28), with 5 minutes for shortcuts, Q&A, and wrap-up (Slides 29–34).
- If possible, run a live VS Code instance alongside the slides. Live demos of ghost text and Copilot Chat are far more impactful than static screenshots.
- The most common sticking point is the GitHub sign-in step (Slide 22). Have a backup plan if authentication is blocked by a corporate firewall.
