# Intro to VS Code and GitHub Copilot — Setup Instructions

> **Track:** Developer | **Level:** Beginner

## Prerequisites

- [ ] VS Code installed (latest stable version)
- [ ] A GitHub account with a Copilot license (Free, Pro, Business, or Enterprise)
- [ ] Internet access for GitHub authentication and Copilot cloud inference
- [ ] Git clone of this repository (for viewing the slide deck locally)

## Quick Start

1. Open a terminal in the course folder:
   ```
   cd "Developer/Beginner/Intro-to-VS-Code-and-GitHub-Copilot"
   ```
2. From the repo root, start the local dev server:
   ```
   npm run dev
   ```
3. Open the URL shown in the terminal (typically http://localhost:5173).
4. Navigate to the course folder in the browser.

## Presenting

- Use **arrow keys** or **click** to navigate slides.
- Press **F** for fullscreen (browser native).
- Speaker notes are embedded in each slide and visible in presenter tools.

## Pre-Session Checklist

- [ ] Verify VS Code is installed and opens correctly
- [ ] Verify the GitHub Copilot extension is available in the Extensions Marketplace
- [ ] Verify you can sign in to GitHub from VS Code
- [ ] Prepare a sample project folder to demo inline completions and Chat
- [ ] Test the slide deck in a browser to ensure styles load correctly
