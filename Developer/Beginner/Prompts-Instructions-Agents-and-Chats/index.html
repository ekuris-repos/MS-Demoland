Copilot Primitives - Prompts, Instructions, Skills, and Agents

Learn the building blocks of Copilot's interaction models, when to reach for each one, and how to craft prompts that deliver reliable results

Overview

What We'll Cover

The Four Primitives

  • Prompts — reusable prompt files (.prompt.md) that capture your best questions
  • Instructions — persistent guidelines (.instructions.md) that shape every response
  • Skills — domain knowledge packages agents can invoke on demand
  • Agents — specialized modes that combine instructions, skills, and tools

When to Use What

One-off task?

Write a prompt — quick, disposable, and shareable with your team.

Ongoing guidance?

Use instructions — they apply automatically to every conversation in your workspace.

Crafting Effective Prompts

  • Be specific about the output format you want (code, list, explanation)
  • Provide context — reference files, constraints, and examples
  • Iterate — refine your prompt based on the response and save what works
